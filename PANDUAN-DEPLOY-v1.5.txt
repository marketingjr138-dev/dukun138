PANDUAN CEPAT DEPLOY v1.5

1. Upload isi folder ini ke root repo GitHub.
2. Pastikan `index.html` langsung kelihatan di root repo, bukan di dalam folder dobel.
3. Cloudflare Pages:
   - Framework preset: None
   - Build command: kosong
   - Build output directory: .
4. Apps Script:
   - Paste file `google-apps-script-v1.5.gs`
   - Run setupSheet()
   - Deploy Web App
   - Copy URL ke Owner Settings PWA.
5. Untuk update global:
   - Edit Google Sheet langsung, atau
   - Edit Owner Settings lalu klik Push ke Google Sheet.
6. Member cukup refresh/buka ulang PWA.

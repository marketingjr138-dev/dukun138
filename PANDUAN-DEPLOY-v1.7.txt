PANDUAN CEPAT DEPLOY v1.7

1. Upload/replace semua file ke root repo GitHub.
2. Cloudflare Pages:
   - Framework preset: None
   - Build command: kosong
   - Build output directory: .
   - Root directory: kosong
3. Apps Script:
   - Pakai file google-apps-script-v1.7.gs
   - Paste ke Apps Script lama
   - Save
   - Deploy > Manage deployments > Edit > New version > Deploy
4. Buka PWA > Admin:
   - Isi Google Apps Script API URL
   - Upload logo/banner/gambar/video dari galeri
   - Klik Push ke Google Sheet
5. Ganti PIN wajib isi PIN lama + PIN baru + ulangi PIN baru.

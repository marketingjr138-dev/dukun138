PANDUAN CEPAT DEPLOY v1.8.4

Fokus: Sync Bootstrap Fix.

Masalah:
Device lain/incognito belum sync karena tidak tahu Google Apps Script API URL.

Cara pakai:
1. Upload/replace file v1.8.4 ke GitHub.
2. Cloudflare Pages: Framework None, Build command kosong, Build output directory .
3. Apps Script: pakai google-apps-script-v1.8.4.gs kalau ingin versi sama.
4. Buka admin di device owner.
5. Isi Google Apps Script API URL.
6. Isi link daftar/login/chat dan setting lain.
7. Klik Simpan.
8. Klik Push ke Google Sheet.
9. Klik Buat Link Share Sinkron.
10. Bagikan link sinkron itu ke device lain/member.

Catatan penting:
Kalau pakai link biasa tanpa ?sync=..., device baru belum tentu tahu endpoint Sheet.
Kalau ingin link biasa langsung sync, gasApiUrl harus diisi permanen di config.js/config.json sebelum deploy.

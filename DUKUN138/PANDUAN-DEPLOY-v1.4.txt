PANDUAN CEPAT DEPLOY v1.4

1. Deploy PWA:
   - Upload semua file ke Cloudflare Pages / hosting.
   - Pastikan _redirects ikut terupload.

2. Buat Google Sheet:
   - Buka sheets.new
   - Extensions > Apps Script
   - Paste isi file google-apps-script-v1.4.gs
   - Save
   - Run setupSheet() sekali dan izinkan akses.

3. Deploy Apps Script:
   - Deploy > New deployment
   - Type: Web app
   - Execute as: Me
   - Who has access: Anyone
   - Copy Web App URL.

4. Hubungkan ke PWA:
   - Buka PWA
   - Masuk Owner Settings
   - Isi Google Apps Script API URL
   - Klik Pull Config Online / Push ke Google Sheet.

5. Cara update global:
   - Edit langsung di Google Sheet, atau edit di Owner Settings lalu klik Push ke Google Sheet.
   - Member refresh/buka ulang PWA untuk melihat update.

PANDUAN CEPAT DEPLOY v1.8.9

Fokus: Static Video Assets Mode.

Tujuan:
Video tutorial bisa pakai file MP4 langsung dari repo agar native player rapi, bisa loop, dan tidak ada overlay Google Drive.

Upload video ke folder:
assets/videos/

Nama file wajib:
- tutorial-daftar.mp4
- tutorial-deposit.mp4
- tutorial-transfer.mp4
- tutorial-withdraw.mp4
- tutorial-promo.mp4

Rekomendasi video:
- 720x1280
- MP4 H.264
- 30 FPS
- 1.5 sampai 2 Mbps
- 10-20 MB per file

Langkah deploy:
1. Upload/replace file v1.8.9 ke GitHub.
2. Upload video MP4 ke assets/videos/ dengan nama wajib di atas.
3. Tunggu Cloudflare deploy sukses.
4. Clear site data/incognito.
5. Buka halaman dan cek tiap tab panduan.

Catatan:
- Static video repo jadi sumber utama.
- Kalau file asset belum ada, video akan error/empty atau fallback ke video Google Drive jika masih tersimpan di config.
- Upload galeri/Google Drive dari admin tetap bisa jadi cadangan, tapi untuk tampilan paling rapi gunakan video repo.
